# PROJECT_STRUCTURE

## Arbol del proyecto
```
Tarea8/
  APUNTES.md
  Apuntes_libro-de-texto_aa0804_Greedy_TSP.pdf
  SPEC.md
  Tarea8.mac
```

## Descripcion de carpetas
- Raiz contiene todos los archivos; no hay subcarpetas.

## Archivos importantes
- APUNTES.md: resumen teorico del TSP, representaciones de grafo.
- SPEC.md: enunciado oficial, requisitos, ejemplos de entrada/salida.
- Tarea8.mac: archivo de entrega; actualmente solo cabecera.
- Apuntes_libro-de-texto_aa0804_Greedy_TSP.pdf: material de apoyo sobre greedy TSP.

## Proposito de componentes relevantes
- SPEC.md define el contrato de funciones `tspPermu` y `tspGreedy`.
- APUNTES.md apoya con contexto y ejemplo de matriz/lista de adyacencia.
- Tarea8.mac es el unico archivo a programar y entregar.
